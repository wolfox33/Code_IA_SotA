# code-ia-sota

Agent scaffold installer for `AGENTS.md` and `.agents`.

## Usage

```sh
npx code-ia-sota
```

Installs the canonical `AGENTS.md` and `.agents/` scaffold into the current directory.

## Local development

```sh
npm install
npm link
code-ia-sota
```

## Package test

```sh
npm pack
npx ./code-ia-sota-0.1.0.tgz
```
