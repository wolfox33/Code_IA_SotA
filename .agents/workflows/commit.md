---
description: Gera um resumo e descrição para commits no Git
---

1. Verificar mudanças com `git status --short`.
2. Revisar `git diff --` e `git diff --staged --`.
3. Agrupar alterações relacionadas em commit lógico.
4. Gerar título em Conventional Commits.
5. Gerar descrição curta (o que/por que/risco).
6. Confirmar antes de `git add` e `git commit`.
7. Executar commit e mostrar `git log -1 --oneline`.
