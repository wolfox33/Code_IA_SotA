# Verification Before Completion — Pitfalls

Conteúdo referencial de pitfalls para verification-before-completion.

## Pitfalls

- Não listar comandos que não foram executados
- Não esconder teste que falhou
- Não finalizar com working tree sujo sem avisar, quando a tarefa pedia commit
- Não transformar checklist em texto longo quando a tarefa é simples
